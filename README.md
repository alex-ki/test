# Run

## 1. Install Node.js

Install __Node.js__ (together with the NPM tool) by following instructions available on [nodejs.org](https://nodejs.org).

## 2. Install Nightwatch

Run 'npm install'

## 3. Run Nightwatch
Run 'nigtwatch'
